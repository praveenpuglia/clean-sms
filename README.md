# Clean SMS

A privacy-focused, clutter-free SMS client for Android that keeps your inbox organized and your OTPs easily accessible.

## Features

- **Automatic Message Organization** – Messages are intelligently categorized for a cleaner inbox
- **Dedicated OTP Tab** – Quickly find and copy one-time passwords without searching through conversations
- **Unread Message Indicators** – Stay on top of new messages with clear visual badges
- **Spam Detection** – Identify potential spam messages at a glance
- **Material Design 3** – Modern, beautiful interface following the latest Material Design guidelines
- **Dynamic Colors** – UI adapts to your device's wallpaper and color scheme (Android 12+)
- **Light & Dark Themes** – Seamless theme switching with system-wide settings
- **Privacy-First** – All processing happens on your device; no data leaves your phone

## Requirements

- Android 13 (API 33) or higher

## Installation

Download the latest APK from the [Releases](https://github.com/praveenpuglia/clean-sms/releases) page.

## Building from Source

```bash
# Clone the repository
git clone https://github.com/praveenpuglia/clean-sms.git
cd clean-sms

# Build release APK
./gradlew assembleRelease
```

The APK will be available at `app/build/outputs/apk/release/app-release.apk`
